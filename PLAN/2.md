# PLAN/2 — Paketleme + Klasör Birleştirme + Import Codemod

**Bağımlılık:** PLAN/1. **Paralellik:** pilot solo; toplu faz modül-başına seri (her modülden sonra build-yeşil). Bağımsız yaprak modüller paralel agent'lere bölünebilir ama her biri build-yeşil bırakmalı.
**Önce oku:** `PLAN/0.md`, `PLAN/1.md`.

## Amaç
Her modülü tek `@nb/<id>` paketine dönüştür: `server/` + `ui/` + `hooks/` yapısı, `module.json.requires`'tan üretilen paket deps, ve tüm import'ların `@nb/*`'a yeniden yazımı. `modules_next/` ortadan kalkar.

## Yapılacaklar
1. **`scripts/seed-package-json.mjs`** (tek seferlik): her modül için `package.json` üret —
   - `name: "@nb/<id>"`, `version: "0.0.0"`, `private: true`.
   - subpath `exports`: `./server`, `./ui`, `./hooks`, `./module.json`.
   - `dependencies`: **`module.json.dependencies.requires`'tan** → her `<dep>` için `"@nb/<dep>": "*"`.
   - ui/hooks olan modüllerde `peerDependencies: { react: "*", next: "*" }`.
2. **Klasör taşıma:**
   - `modules/<id>/*` iş-mantığı dosyaları → `modules/<id>/server/`.
   - `modules_next/<id>/{ui,hooks}` → `modules/<id>/{ui,hooks}`; `*.service.next.ts` → `server/`.
   - `entities/` → `server/entities/`.
   - Her alt-klasöre **dar** `index.ts` barrel (döngü riskini azalt).
3. **`scripts/migrate-to-packages.mjs`** (ts-morph veya jscodeshift, tek seferlik) import rewrite:
   - `@/modules_next/<id>/ui/X` → `@nb/<id>/ui`
   - `@/modules_next/<id>/hooks/X` → `@nb/<id>/hooks`
   - `@/modules/<id>/<f>` → `@nb/<id>/server`
   - Modül-içi göreli yolları `server/`/`ui/` taşınmasına göre düzelt.
   - `app/` katmanı `@/*` kalır ama modül import'ları `@nb/*`'a döner.
4. **Pilot gate:** PLAN/1 ile aynı yaprak modülü uçtan uca yap → `npm install` + `npm run build` + `npm test` yeşil; TypeORM entity yükleme + bir migration dene. **Kanıtlanmadan toplu migrasyona geçme.**

## Bitti tanımı (DoD)
- Tüm 78 modül `@nb/<id>` paketi; `modules_next/` boş/silinmiş.
- `npm run build` + `npm test` yeşil.

## Dokunulan dosyalar
78× `modules/<id>/package.json` (yeni), ~1574 dosya (taşıma + import), `scripts/seed-package-json.mjs` + `scripts/migrate-to-packages.mjs` (yeni, tek seferlik).

## Risk
Import rewrite en riskli mekanik adım → **pilot + modül-başına build-yeşil + codemod** ile de-risk. Barrel'ları dar tut (dosya-düzeyi döngü). TypeORM reflect-metadata paket sınırında pilotla doğrula.
