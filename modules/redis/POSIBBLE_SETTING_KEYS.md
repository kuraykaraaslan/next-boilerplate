# Possible Setting Keys

| Key | Source | Notes |
| --- | --- | --- |
| env:REDIS_HOST | redis.service.ts, redis.bullmq.ts | Redis host |
| env:REDIS_PORT | redis.service.ts, redis.bullmq.ts | Redis port |
| env:REDIS_PASSWORD | redis.service.ts, redis.bullmq.ts | Redis password |
| cacheJitterFactor | redis.cache.ts | candidate; jitter factor default 0.1 |
